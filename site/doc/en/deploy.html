Deploying a Brython application on a server

For deployment on a web server accessible to users of your application, you don't have to install all the development environment. In the downloads page, choose one of the archives (zip, gz or bz2) called Brython-YYYYMMDD-HHMMSS ; unpack it and upload its content to the directory where you want to install your Brython application. It only holds the Brython distribution : brython.js and the built-in libraries in directories libs and Lib
